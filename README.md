# Eventora

Eventora is a premium event discovery and booking platform built by `Seniorman Computers`.

It is designed around elegant event presentation, refined reservations, guest-facing booking management, and organizer-side operational visibility.

## Overview

- Luxury-styled landing page for event discovery
- Event listing experience with curated categories and booking status
- Featured event detail page with reservation tiers and highlights
- Attendee dashboard for reservations, passes, and saved experiences
- Organizer dashboard for sales visibility, seat demand, and operations tracking
